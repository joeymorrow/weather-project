# Buddy Control Panel (Admin)

The Control Panel is used by location managers to control signage, alerts, and branding.

## Key Controls
*   **Beacon Flare:** Instantly takes over the dashboard with a full-screen, pulsing emergency alert. Useful for immediate, severe local warnings (e.g., Tornado, Lockdown).
*   **Signage Playlist:** Add text, image, video, or Canva/Web iframe slides. These will rotate dynamically with the main Buddy view based on their set durations.
*   **School & Location Alerts:** Trigger pre-defined EAP (Emergency Action Plan) protocols (like "I Love U Guys" SRP) for specific dynamic endpoints.
*   **Main Config & Branding:** Update the base query location and the "Powered By" text at the bottom left of the screen.
