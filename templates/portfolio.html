Joey Morrow A Project-Based Portfolio

What started on May 3rd as a simple Python script to determine if I should wear waterproof boots or nice shoes in the melting spring snow evolved in just 7 days into BEACON Buddy. It is now a resilient, CI/CD-managed local weather, scheduling, and emergency response dashboard. This portfolio illustrates the practical application of advanced software engineering principles, from low-level systems architecture to scalable cloud infrastructure.

Infrastructure, Architecture & Systems Engineering

Worker Concurrency & POSIX File Locks

Solved OS-level race conditions by implementing fcntl POSIX file locks, ensuring that only one Gunicorn worker could spawn background threads, preventing redundant, resource-intensive loops in a multi-process production environment.

UDP Multicast & Emergency Protocols

Wrote low-level Python socket logic (socket.IP_ADD_MEMBERSHIP) to listen for raw UDP multicast packets, integrating the "I Love U Guys" Standard Response Protocol for real-time emergency alerts on digital signage.

Secure IPC with D-Bus

Implemented robust Inter-Process Communication (IPC) utilizing D-Bus to securely map hardware signals and system services directly to isolated containerized dashboards without exposing bare-metal network ports.

Resilient Edge CI/CD

Architected a self-hosted GitHub Actions pipeline featuring automated headless DOM testing (Puppeteer), Static and Dynamic Application Security Testing (SAST/DAST), comprehensive secret leak detection, and zero-downtime Docker Compose edge delivery to maintain robust operational security.

Interactive Engines, WebGL & UX

Demoscene Bitwise Rendering

Bypassed the CPU to manipulate the GPU's memory array directly using bitwise shifts (>>, <<) and XOR (^) on a Unix timestamp, creating zero-cost 24-bit RGB strobe effects for the "Rave" Easter egg.

3D Engine Memory Optimization

Implemented zero-allocation particle pools, pre-allocated THREE.Vector3 variables, and an aggressive idle throttler to drop WebGL to 1 FPS when out of context, preventing browser crashes and ensuring smooth 60fps performance.

GPU Geometry Instancing

Compressed over 650 individual draw calls for trees, clouds, and mountains into single THREE.InstancedMesh calls, dramatically reducing CPU bottlenecks and enabling a vast, animated 3D world.

Responsive Mobile & Multi-Touch UI

Mastered 100dvh and CSS clamp() for fluid layouts and tracked e.changedTouches[i].identifier to allow independent thumb controls for the joystick and camera, ensuring a seamless experience on any device.
